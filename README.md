\# 🤖 StudyMind AI — Student Productivity Assistant



An AI-powered Telegram bot built for Indian college students (BBA/MBA) to manage assignments, exams, internships and study smarter.



\## 🚀 Features

\- 📚 Assignment \& Exam Tracking

\- ⏰ Automatic 24h Deadline Reminders

\- 🤖 AI Study Summaries (Gemini 2.5 Flash)

\- 📅 AI Daily Study Planner

\- 🧠 AI Practice Quiz Generator

\- 💪 Personalized Motivation Messages

\- 📖 AI Exam Strategy Tips

\- 💼 Live Internship Alerts (Internshala)

\- 🏆 Hackathon Finder



\## 🛠️ Tech Stack

\- Python 3.13

\- python-telegram-bot

\- Google Gemini 2.5 Flash API

\- BeautifulSoup4

\- APScheduler



\## 📱 Bot Commands

| Command | Description |

|---------|-------------|

| /add | Add assignment with deadline |

| /exam | Add exam with date |

| /list | View all tasks |

| /done | Mark task completed |

| /countdown | Exam countdown timer |

| /summarize | AI study summary |

| /plan | AI daily study plan |

| /quiz | AI practice questions |

| /motivate | Personalized motivation |

| /tips | AI exam strategy |

| /internships | Live internship alerts |

| /hackathons | Find hackathons |



\## ⚙️ Setup

1\. Clone this repo

2\. Install dependencies:

pip install python-telegram-bot google-genai requests beautifulsoup4 apscheduler

3\. Create config.py with your tokens:

TOKEN = "your\_telegram\_token"

GEMINI\_API\_KEY = "your\_gemini\_key"

4\. Run:

python bot.py
